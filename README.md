# Silver Price Calculator & Silver Sales Dashboard

Streamlit app for analyzing historical silver prices in India and visualizing
state-wise silver purchases. Includes a price calculator with currency conversion.

## Run locally

1. Create and activate a virtual environment.
2. Install dependencies from `pyproject.toml`.
3. Start the app:

   ```sh
   streamlit run streamlit_app.py
   ```

## Data files

Place these CSVs in the workspace root (one level above this app folder):

- `historical_silver_price.csv`
- `state_wise_silver_purchased_kg.csv`
